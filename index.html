온브릭 유한책임회사 설립

2020년 9월 17일 법인 설립 등기 신청

Intro

더 높은 효율을 발휘하도록 돕는 App 을 개발합니다.

가벼운 유틸리티 앱부터, 업무에서 사용할 수 있는 앱까지, 온브릭의 앱은 마이크로소프트 스토어에 있습니다

Assembly Information

닷넷 어셈블리의 주요 속성을 한눈에 표시합니다

  • 폴더를 지정하여, 모든 어셈블리의 주요 속성을 목록으로 표시합니다.
  • 서비스 중인 dll 이 최적화 빌드인지 확인할 수 있습니다.

Sheet Converter

엑셀 파일과 CSV 파일 사이의 저장을 위한 작업을 단순화합니다

  • 엑셀 파일의 여러 시트를 각각 CSV 파일로 저장합니다.
  • 여러개의 CSV 파일을 각각의 시트로 매핑하여 하나의 엑셀 파일로 저장합니다.

Meeting Minutes

회의록 정리를 위한 개인 환경을 제공합니다.

  • 연락처를 관리합니다.
  • 회의록을 관리합니다.
  • 회의와 관련된 참석자와 할 일을 관리합니다.

StoredProcedure Factory

SQL Server 의 Stored procedure 스크립트를 자동으로 생성합니다.

  • SQL Server 로의 연결 문자열을 저장합니다.
  • Stored procedure 이름 규칙을 지정합니다.
  • SQL Server 의 Table 을 조회합니다.
  • INSERT, SELECT, UPDATE 하는 stored procedure 스크립트를 만듭니다.

Entity Factory

SQL Server 의 데이터 테이블과 1:1 매핑되는 클래스 코드를 자동으로 생성합니다.

  • SQL Server 로의 연결 문자열을 저장합니다.
  • meta data 를 조회하여 테이블과 컬럼 정보를 조회합니다.
  • 조회된 정보를 이용하여 PONO 클래스 코드를 생성합니다.
  • 생성된 클래스 소스 코드를 파일로 저장합니다.

SourceCode Factory

엑셀 파일에 정리된 설계로, 소스코드 포워드 엔지니어링을 제공합니다.

  • 엑셀 템플릿 파일을 로컬에 저장합니다.
  • 편집한 템플릿 파일을 앱에 등록합니다.
  • 등록한 정보를 바탕으로 소스 코드를 미리보기합니다.
  • 대상 폴더를 지정하여 소스 코드 파일을 저장합니다.

UnitTest Factory

소스코드를 분석하여 단위 테스트 코드(xUnit, NUnit, MSTest)를 스케폴딩합니다.

  • C# 소스코드를 선택하여 클래스와 메서드를 분석합니다.
  • 사용자가 선택한 라이브러리를 기준으로 단위 테스트 코드를 생성합니다.
  • 대상 폴더를 지정하여 단위 테스트 코드를 저장합니다.

Resource Factory

영어, 한국어, 중국어, 일본어를 한번에 편집하여 리소스 파일을 생성합니다.

  • 템플릿 파일을 저장합니다.
  • 편집한 템플릿 파일을 읽어와 목록으로 표시합니다.
  • 리소스 파일 타입(resw, resw)과 구조를 선택하여 로컬에 저장합니다.
